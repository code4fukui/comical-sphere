# comical-sphere

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A project that displays animated 3D spheres with comical images.

## Demo
https://code4fukui.github.io/comical-sphere/

## Features
- Displays multiple animated 3D spheres
- Spheres rotate around a circular path
- Textures are comical images

## Usage
1. Open the provided HTML file in a web browser.
2. Use the arrow keys to move the camera view.
3. Use the mouse wheel to zoom in and out.
