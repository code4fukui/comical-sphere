# comical-sphere

3枚のコミカルなイメージをWebブラウザ上で3Dスフィアに配置し、カメラ操作やマウスホイールでスフィアの表示を楽しめるWebアプリケーションです。

## デモ
https://code4fukui.github.io/comical-sphere/

## 機能
- 3枚のコミカルなイメージをスフィアに配置
- カメラ操作(矢印キー)でスフィアを回転して表示
- マウスホイールでスフィアの距離を調整

## 使い方
Webブラウザで上記デモページを開き、矢印キーやマウスホイールを使ってスフィアを操作してください。
